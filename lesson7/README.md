# Supervised Learning: Basic Concepts

## 1. Chiziqli regressiya va klassifikatsiya

| Tushuncha | Ta'rif |
|-----------|--------|
| **Chiziqli regressiya** | Uzluksiz (continuous) qiymatni bashorat qilish usuli. Ma'lumotlar orasidan eng mos to'g'ri chiziq topadi. Formula: `y = mx + b` |
| **Klassifikatsiya** | Ma'lumotni guruhlarga (kategoriyalarga) ajratish usuli. Natija: diskret qiymat (Ha/Yo'q, Spam/Spam emas) |

**Misol:**
- Regressiya → Uy narxini bashorat qilish (300 mln, 450 mln...)
- Klassifikatsiya → Email spam yoki yo'qligini aniqlash (0 yoki 1)

## 2. Regressiya va Klassifikatsiya farqlari

| Xususiyat | Regressiya | Klassifikatsiya |
|-----------|-----------|----------------|
| **Natija turi** | Uzluksiz son (150.5, 300000) | Kategoriya (0/1, Ha/Yo'q) |
| **Maqsad** | Qiymatni bashorat qilish | Guruhga ajratish |
| **Algoritmlar** | Linear Regression | Logistic Regression, KNN |
| **Baholash** | MSE, R² | Accuracy, Precision |
| **Grafik** | Chiziqli grafik | Chegaraviy chiziq |
| **Misol** | Haroratni bashorat | Kasallikni aniqlash |
