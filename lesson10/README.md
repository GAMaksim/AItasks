## 2. Computer Vision nima?

**Computer Vision** — kompyuterga rasm va videolarni "ko'rish" va tushunish qobiliyatini beruvchi AI tarmog'i.

**Qanday ishlaydi:**
1. **Rasmni olish** — kamera yoki fayl
2. **Piksellarni tahlil** — ranglar, yorug'lik, qirralar
3. **Feature extraction** — muhim xususiyatlarni ajratish (CNN)
4. **Klassifikatsiya** — "Bu mushuk", "Bu mashina" deb aniqlash

**Ishlatilishi:** Yuz tanish, avtopilot, tibbiy rasm tahlili, QR kod o'qish
