# TensorFlow.js va Modellar

## 1. TensorFlow.js nima?

**TensorFlow.js** — brauzerda va Node.js da to'g'ridan-to'g'ri AI/ML modellarni ishlatish imkonini beruvchi **JavaScript kutubxonasi**.

| Xususiyat | Tavsif |
|-----------|--------|
| **Brauzerde ishlaydi** | Hech narsa o'rnatmasdan, faqat HTML+JS bilan |
| **GPU tezlashtirish** | WebGL orqali GPU dan foydalanadi |
| **Tayyor modellar** | MobileNet, COCO-SSD, DeepLab va boshqalar |
| **O'qitish** | Brauzerda model o'qitish ham mumkin |

## 2. TensorFlow.js modellari

| Model | Vazifasi | Tavsif |
|-------|---------|--------|
| **MobileNet** | Image Classification | Rasmni aniqlash (mushuk, it, mashina) |
| **COCO-SSD** | Object Detection | Rasmdagi obyektlarni topish va ramka chizish |
| **DeepLab** | Image Segmentation | Rasmni segmentlarga bo'lish (odam, fon) |
| **PoseNet** | Pose Estimation | Tananing holatini aniqlash |
| **Toxicity** | Text Classification | Yomon so'zlarni aniqlash |
